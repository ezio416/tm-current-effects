# Current Effects
Displays current effects on your Trackmania 2020 car.

![](image_text.png)

## Currently working:
- Engine Off
- Forced Acceleration
- No Brakes
- No Grip
- No Steering
- Reactor Boost (yellow/red, up/down, last-second timer)
- Slow-Mo (all 4 levels)
- Turbo (all 5 levels, timer)
- Editor playtest
- Viewing records (kind of)
- Online

## Still needed:
