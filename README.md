# Current Effects
Openplanet plugin for Trackmania 2020 to display current effects on the car

## Currently working:
- Engine Off
- Forced Acceleration
- No Brakes
- No Grip
- No Steering
- Reactor Boost (red/yellow + up/down)
- Slow-Mo (levels 1 + 2)
- Turbo (yes/no)

## Still needed:
