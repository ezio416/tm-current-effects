# Current Effects
Displays current effects on your Trackmania 2020 car.

## Currently working:
- Engine Off
- Forced Acceleration
- No Brakes
- No Grip
- No Steering
- Reactor Boost (red/yellow + up/down)
- Slow-Mo (levels 1 + 2)
- Turbo (yes/no)
- Editor playtest
- Viewing records (kind of)
- Online
